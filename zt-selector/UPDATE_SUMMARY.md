## ✨ 新增功能

### 功能亮点

1. **双RSI信号分析**
   - 🔵 快速RSI(13) - 对价格变化更敏感
   - 🔴 慢速RSI(42) - 趋势确认
   - 📊 差值阈值：±16（日线级别）

2. **交易信号**
   - 🟢 **做多信号**：两RSI同时下降 + 差值约-16
   - 🔴 **做空信号**：两RSI同时上升 + 差值约+16
   - ⚪ **观望**：其他情况

3. **批量分析**
   - 支持分析选股结果CSV
   - 支持自定义股票列表
   - 自动生成分析报告

4. **详细指标**
   - RSI快慢线数值
   - RSI差值
   - 趋势方向（上升/下降/震荡）
   - 信号分布统计

## ⚙️ 技术参数

| 参数 | 值 | 说明 |
|------|-----|------|
| 快速RSI周期 | 13 | 对价格变化更敏感 |
| 慢速RSI周期 | 42 | 趋势确认指标 |
| 差值阈值（日线） | ±16 | 触发信号的最小差值 |
| 趋势回看周期 | 3 | 判断趋势的K线数 |
| 默认分析周期 | 日线 | 适合A股日线分析 |
