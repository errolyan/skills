# 涨停敢死队选股技能 (zt-selector)

基于akshare数据源的A股模拟盘选股工具

## 功能特性

- ✅ 6大规则选股系统
- ✅ 智能过滤（科创板、ST、超大市值）
- ✅ 实时分析上证指数趋势
- ✅ 识别热门板块（板块共振效应）
- ✅ K线技术面分析（MA均线多头排列）
- ✅ 近5日红肥绿瘦检查
- ✅ 涨停历史统计
- ✅ 市盈率展示
- ✅ 自动生成CSV报告

## 触发条件

当用户提到以下关键词时，该技能会自动触发：
- 涨停选股
- 敢死队选股
- A股选股
- 尾盘选股
- 涨停板选股
- 今日选股
- zt-selector

## 使用方法

只需要对Claude说：
```
帮我选今日涨停股票
```

或者：
```
尾盘选股，找一些符合涨停条件的股票
```

Claude会自动：
1. 安装必要的依赖（akshare、pandas）
2. 生成并执行选股脚本
3. 分析大盘和板块
4. 输出符合条件的股票列表
5. 给出重点关注建议和风险提示
