# 涨停敢死队选股技能 + 双RSI技术分析 (zt-selector)

基于akshare数据源的A股模拟盘选股工具 + RSI技术信号分析器

## 功能特性

### 功能一：涨停板选股
- ✅ 6大规则选股系统
- ✅ 智能过滤（科创板、ST、超大市值）
- ✅ 实时分析上证指数趋势
- ✅ 识别热门板块（板块共振效应）
- ✅ K线技术面分析（MA均线多头排列）
- ✅ 近5日红肥绿瘦检查
- ✅ 涨停历史统计
- ✅ 市盈率展示
- ✅ 自动生成CSV报告

### 功能二：双RSI技术分析 🆕
- ✅ RSI(13) + RSI(42)离散策略
- ✅ 自动生成做多/做空/观望信号
- ✅ 支持批量分析股票
- ✅ 可直接分析选股结果CSV
- ✅ 日线级别技术分析
- ✅ 详细的RSI趋势判断
- ✅ 自动保存分析报告

## 触发条件

当用户提到以下关键词时，该技能会自动触发：

**涨停选股相关**：
- 涨停选股
- 敢死队选股
- A股选股
- 尾盘选股
- 涨停板选股
- 今日选股

**RSI分析相关** 🆕：
- RSI分析
- 双RSI
- 技术信号
- RSI信号
- 技术分析

或直接使用：`zt-selector`

## 使用方法

### 方式一：涨停板选股

只需要对Claude说：
```
帮我选今日涨停股票
```

或者：
```
尾盘选股，找一些符合涨停条件的股票
```

### 方式二：RSI技术分析 🆕

对已有的选股结果进行RSI分析：
```
对这些股票进行RSI技术分析
```

或者：
```
分析这只股票的RSI信号
```

### 方式三：组合使用 🆕

先选股，再进行RSI分析：
```
帮我选股，然后对结果进行RSI技术分析
```

Claude会自动：
1. 安装必要的依赖（akshare、pandas）
2. 生成并执行选股脚本
3. 分析大盘和板块
4. 输出符合条件的股票列表
5. 给出重点关注建议和风险提示
